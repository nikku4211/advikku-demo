# Advikku Demo
Demo for a WIP music engine for the Game Boy Advance that uses a cell-based custom format.

## Building

### Dependencies
- Tonclib (I plan to remove this dependency later)
- DevkitARM (Which includes Tonclib)
- PTexConv
